INTRODUCTION                                    *rspec-simple*

This plugin allows you to easily run the rspec file current in the buffer via
bundle exec.

*RSpec*
RSpec       Given a line number, run bundle exec rspec with that line.

*<leader>ft*
<leader>ft    Run bundle exec rspec with the line under the cursor. Short for 'focus test'.

*<leader>at*
<leader>at    Run bundle exec rspec on the current file. Short for 'all tests'.
